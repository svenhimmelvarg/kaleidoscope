
## Intro 

Kaleidescope reads generates from the following folder 

    $COMFYUI_INSTANCE_BASE_PATH/output/release 

and makes the workflow available through the search interface. Let's work through an example 

When you first start your page will look like this:

## Preparing a Workflow 

Go to the menu and browse templates and chose a text to image workflow. I have chosen ZImageTurbo. 

<img width="558" height="537" alt="image" src="https://github.com/user-attachments/assets/87787bd6-1500-41a8-a901-1de608c4bc71" />

***Important***  - update the save location  to release/ANYTHING/YOU/WANT/AFTER/THIS 
It's important this points to a subfolder in your release folder so that indexer can find it.
